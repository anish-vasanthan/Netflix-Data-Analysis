# 📊 Netflix Data Analysis Project

## 📌 Overview
This project focuses on analyzing the **Netflix dataset** using Python to extract meaningful insights through **data cleaning, exploration, visualization, and hypothesis testing**.

The objective of this mini project is to understand content trends, distribution patterns, and useful insights from the Netflix dataset using **Exploratory Data Analysis (EDA)** techniques.

---

## 🎯 Project Objectives
- Clean and preprocess the Netflix dataset
- Explore the dataset and understand its structure
- Perform Exploratory Data Analysis (EDA)
- Visualize important trends and patterns
- Conduct basic statistical and hypothesis testing
- Present findings in a clear and understandable format

---

## 🛠️ Tools & Technologies Used
- **Python**
- **Pandas**
- **NumPy**
- **Matplotlib**
- **Seaborn**
- **Jupyter Notebook / Google Colab**

---

## 📂 Project Workflow

### 1. Data Cleaning
- Checked missing values
- Removed duplicates
- Handled null values
- Standardized columns where necessary

### 2. Data Exploration
- Understood dataset structure
- Checked data types
- Explored categories like:
  - Content Type
  - Country
  - Rating
  - Release Year
  - Duration

### 3. Exploratory Data Analysis (EDA)
Performed analysis to answer questions such as:
- What type of content is more common on Netflix?
- Which countries produce the most Netflix content?
- How has content release changed over the years?
- Which ratings are most frequent?
- What patterns can be identified from the dataset?

### 4. Data Visualization
Created various charts and graphs to represent insights visually, including:
- Bar Charts
- Count Plots
- Pie Charts
- Histograms
- Line Charts
- Heatmaps (if used)

### 5. Hypothesis Testing
Performed statistical testing to validate assumptions and identify relationships in the dataset.

---

## 📈 Key Insights
Some important insights observed from the project:
- Movies dominate the Netflix platform compared to TV Shows
- Certain countries contribute significantly more content
- Content addition increased in specific years
- Popular ratings appear more frequently across titles
- Visual trends helped identify patterns in Netflix content distribution
